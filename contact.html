How does this javascript accordian work?

A const is created that grabs the divs with the faq class. Then JS listens for whenever that div is clicked and, using the classList property, toggles the active class for whichever faq was clicked on.

What other classList properties exist?

There are 3 common properties associated with classList. The add() property will add to your selected classr, remove() takes it away, and toggle() will add or remove it depending on its current state.
